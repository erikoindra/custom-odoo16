Postponed VAT

Postponed VAT is a core Odoo module developed to automatically validate a reversed Invoice/VB when validating Invoice/VB.


Features

Tax as a Postponed VAT

This feature allows you to define which tax is a Postponed VAT.

Auto Validate Reversed Invoice/VB

Postponed VAT allows automatically create and validate the reversed move for Invoice/VB that has at least a product/line with postponed VAT.


How to use Postponed VAT

  • Download & install the Postponed VAT module
  • Go to Accounting → Taxes → Create or Edit existing Tax → Select the 'Is a Postponed VAT' box.
  • Create or Select Existing Invoice/VB and add postponed VAT to the lines.
  • Confirm or Validate the Invoice/VB and it will automatically create a reversed move for the postponed VAT amounts.

Why you should use Postponed VAT

Increased efficiency

You can save time and effort by automating the credit/bill note process.

Greater budget management

Allows predefined configuration for postponed VAT, ensuring that invoices are in their current products VAT condition.
